# Part 1 project

## Configuring the environment
1. Add these packages with npm install (joi, mongodb, mongoose)
2. Configure the dbpath of mongodb with the address of the data folder to store the db in folder

## Run the server
1. Run npm start 
2. Acess localhost:3000

## Run the Tests
1. To run the test use Postman
2. Acess localhost:3000/users to test GET and POST methods
3. Acess localhost:3000/users/:id to test GET, PUT and DELETE methods
